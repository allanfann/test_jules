# Gemini 開發環境

本文件概述了 Gemini 開發所使用的專案結構和技術。

## 目錄結構

-   `app/`：此目錄包含前端應用程式的程式碼。
-   `backend/`：此目錄包含後端服務的程式碼。

## 前端 (`app/`)

前端是使用 [Flutter](https://flutter.dev/) 建置並與 [Firebase](https://firebase.google.com/) 服務整合的 Web 應用程式。

-   **框架**：Flutter
-   **平台**：Web
-   **後端整合**：Firebase

## 後端 (`backend/`)

後端程式碼存放在此目錄中，為前端應用程式提供必要的 API 和服務。
